Search the SENA knowledge base for relevant patterns, solutions, and insights.

Run this command:
```bash
sena knowledge search "$ARGUMENTS"
```

This searches across SENA's knowledge domains:
- Reasoning frameworks (First Principles, 5 Whys, Systems Thinking)
- Security patterns (OWASP, Auth, Crypto)
- Performance patterns (Big O, Caching, Async)
- Architecture patterns (SOLID, DDD, Design Patterns)

Display the search results to the user with relevance scores.
