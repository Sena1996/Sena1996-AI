Enhanced git operations with SENA's AI-powered formatting and assistance.

## Available Commands

### Show enhanced git status
```bash
sena git status
```

### AI-generated commit message
```bash
sena git commit
```

### Formatted commit history
```bash
sena git log
```

### Show changes
```bash
sena git diff
```

## Features

| Command | Enhancement |
|---------|-------------|
| status | Beautiful Unicode box formatting |
| commit | AI-generated commit messages based on changes |
| log | Formatted, colorized commit history |
| diff | Highlighted diff with context |
