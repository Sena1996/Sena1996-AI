# Homebrew Tap for SENA

Install SENA via Homebrew:

```bash
brew tap sena1996/sena https://github.com/Sena1996/Sena1996-AI
brew install sena
```

Or install directly:

```bash
brew install sena1996/sena/sena
```

## Update

```bash
brew upgrade sena
```

## Uninstall

```bash
brew uninstall sena
```
